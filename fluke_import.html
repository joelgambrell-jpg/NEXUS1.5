Fluke Connect Import

Goal: same pattern as Snap-on ConnecTorq — export a file from the vendor app, then import here to store structured readings and include them in Package Export.
This importer is format-flexible (works even before you own the meter). When you get the device, export a CSV from Fluke Connect and import it here.
If Fluke Connect provides PDF as well, keep using PDF for record attachments, but use CSV for data import.
